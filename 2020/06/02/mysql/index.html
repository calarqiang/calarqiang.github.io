MySql基础学习

SQL的分类

SQL语句主要可以划分为以下三种类型：

  1. DDL (Data Definition Languages）语句 : 数据定义语言，这些语句定义了不同的数据段、数据库、表、列、索引等数据库对象的定义。常用的语句关键字主要包括 create、drop、alter等。
  1. DML (Data Manipulation Language）语句 : 数据操纵语句，用于添加、删除、更新和查询数据库记录，并检查数据完整性，常用的语句关键字主要包括 insert、delete、udpate 和*select *等。(增添改查）
  1. DCL (Data Control Language）语句： 数据控制语句，用于控制不同数据段直接的许可和访问级别的语句。这些语句定义了数据库、表、字段、用户的访问权限和安全级别。主要的语句关键字包括 grant、revoke 等。
  1. TCL （Transaction Control Language）：事务控制语言
    • SAVEPOINT 设置保存点
    • ROLLBACK 回滚
    • START TRANSACTION 开始事务
    • COMMIT 提交事务

Mysql常用命令

  1. 查看所有的数据库：

    show databases;

  2. 打开指定的数据库

    use 数据库名;

  3. 查看当前数据库的所有表

    • 在当前所处数据库下查看

      show tables;

    • 查看其它的数据库中的所有表

      show tables from 表所在的数据库名;

  4. 创建一个表

  5. 查询表结构

    desc 表名

  6. 查看mysql服务器的版本：

    • 登录到mysql服务端之后：

      select version();

    • 没有登录到mysql服务端：

      mysql --version

  7. 查看当前所处的数据库：

    select database();

MySql语法规范

  1. 不区分大小写，但建议关键字大写，表名、列名小写。
  2. 每条命令最好用分号区分
  3. 每条命令根据需要，可以进行缩进或者换行。
  4. 注释：
    • 单行注释： #注释文字 或者-- 注释文字
    • 多行注释：/*注释*/
